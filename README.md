# cc-slides-master

## tl;dr

Make changes in ```slides.md```, then run
```
make
```
to render both ```index.html``` and ```ccc-slides-master.pdf```.


## ditaa diagrams

If you want to use ditaa diagrams, you need a jdk1.8+ and the ditaa project.
The later can be downloaded with

```
make ditaa
```
